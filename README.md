# 🏠 Real Estate Price Predictor

An interactive Streamlit app that predicts property prices based on **size**, **year built**, and whether the property has a **sea view**.

## 🚀 Features
- Linear regression model trained on `real_estate.csv`
- Interactive UI for entering property details
- Instant price predictions
- Model performance metrics (R², RMSE)
- Scatter plot of actual vs predicted prices
